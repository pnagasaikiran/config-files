## Keybinds
```
terminal - ctrl + enter (or) win + shift + t
firefox-browser - ctrl + alt + w
nemo - ctrl + alt + e
moving through windows - win + h,j,k,l
moving windows - win + shift + h,j,k,l
stacking layout - win + s
tabbed layout - win + t
split layout - win + p
open d menu - win + d
open horizontal split - win + e
open vertical split - win + o
make a window floating or tiling - win + shift + space
change focus between tiling and floating windows - win + space
change workspace - win + 1,2,3,4,5,6,7,8,9,0
move window to workspace - win + shift + 1,2,3,4,5,6,7,8,9,0
reload i3 config - win + shift + c
restart i3 - win + shift + r
close a window - win + q
quit i3 - win + shift + esc
resize window - win + r and arrow keys or h,j,k,l to resize windows
exit resize mode - enter (or) escape (or) win + r
open rofi menu - win + m
screenshot of screen - ctrl + print
screenshot of area - ctrl + shift + print
go to parent container - win + y
go to child container - win + shift + y
move scratchpad - win + shift + u
show scratchpad - win + u
```

Rename this file as .bak after you install dunst:-
```
/usr/share/dbus-1/services/org.knopwob.dunst.service
/usr/share/dbus-1/services/org.xfce.xfce4-notifyd.Notifications.service.bak
```
Also after you install brightnessctl, run this command:-
```
sudo chmod +s /usr/bin/brightnessctl
```
